# CRUNCHOS
We are developing a Linux-based operating system focused on reliability, compatibility, and performance.

Development began on May 11, and despite being only 12 years old, we are committed to learning and building the project seriously. AI-assisted development tools will help us throughout the process.

Our primary objective is to create an OS capable of reliably running:

* Linux applications
* Windows 10/11 applications

The operating system will also feature a custom-built browser developed by Enrico.

This project is designed to combine learning, experimentation, and real software development.
